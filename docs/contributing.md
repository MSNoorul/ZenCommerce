# Contributing

Thank you for considering contributing to EcomExpress!

How to contribute
- Fork the repo and create a feature branch
- Ensure tests pass: mvn test or ./gradlew test
- Follow code style and add meaningful tests
- Open a PR with a clear description

Branching model
- main: production-ready
- develop: integration for next release
- feat/xyz: new features

Code of Conduct
- Be respectful, inclusive, and collaborative
